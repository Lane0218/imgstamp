# ImgStamp 产品需求文档 (PRD)

## 1. 项目背景
ImgStamp 是一款面向个人使用的本地桌面端软件，旨在帮助用户高效整理照片，为精选照片添加统一的白色边框及包含日期、地点、描述的文字标注。

## 2. 目标用户与使用场景
- **用户**：个人用户（开发者自己）。
- **场景**：
    - 每年一次的照片整理。
    - 处理来源多样（相机、手机截图、微信图片等）。
    - 需要从大量照片中筛选 100-200 张精华。
    - 最终产出风格统一、带有元数据标注的成品图片。
- **核心价值**：稳定、高效、可复用，减少重复劳动。

## 3. 功能需求

### 3.1 照片导入与管理
- **功能描述**：
    - 支持选择本地文件夹导入照片。
    - 支持常见图片格式（JPG, PNG），暂不支持 HEIC。
    - 列表/网格视图显示照片缩略图。
    - 状态标记：
        - 选中/未选：决定是否纳入处理范围。
        - 状态：待处理/信息完整/已导出。
- **约束**：严禁修改原始图片文件。

### 3.2 单张照片信息编辑
- **拍摄日期**：
    - 自动读取 EXIF 日期（若有）。
    - 支持手动修改/输入（针对截图/微信图）。
    - 支持对多张照片批量设置同一天日期。
- **拍摄地点**：
    - 文本输入（精确到“市”）。
    - 默认填入“湖南长沙”。
    - 支持下拉选择历史地点。
    - 支持对多张照片批量设置地点。
- **描述文本**：
    - 自由文本输入（1行）。
    - 记录人物或事件详情。

### 3.3 批量操作与效率
- **上一张/下一张**：在右侧属性编辑区提供快捷切换按钮。
- **批量填充**：选中多张照片，支持“应用全部（日期/地点/描述）”与“按字段应用”，并直接覆盖所有选中照片。
- **进度可视化**：通过颜色或图标明显区分信息缺失的照片。

### 3.4 成品导出（核心功能）
- **图片处理**：
    - 保持原图比例。
    - 增加外部白色边框。
    - 在边框区域绘制文字（日期、地点、描述）。
    - 文字排版要求整齐、中英文兼容。
- **输出**：
    - 生成新的图片文件。
    - 保存到用户指定的导出目录。
    - 文件名避免冲突。
- **自定义**：
    - 支持设置最终图片尺寸（如 5寸/6寸 冲印标准）。

### 3.5 项目管理
- **项目保存**：
    - 保存当前工作进度（JSON文件或其他格式）。
    - 记录图片路径引用、已填写的元数据、选中状态。
    - **自动保存**：每 1 分钟自动保存一次（需已手动保存过项目文件），导出过程中暂停。
- **项目加载**：
    - 能够打开之前的项目文件，恢复工作现场。

## 4. 非功能需求 (Non-Goals)
- 无云端同步。
- 无账号登录系统。
- 无社交分享功能。
- 无 AI 自动生成内容。
- 无复杂滤镜/修图功能。

## 5. UI/UX 风格
- **风格**：极简、工具化、工程风。
- **重点**：操作流畅，快捷键支持（←/→ 切换、Space 勾选、Ctrl+Enter 导出），信息密度适中。
